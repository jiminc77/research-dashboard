# GitHub Project Setup

This document has been replaced by the milestone-based DGCC Research OS guide.

See:

- `docs/research_ops_manual.html`
- `README.md`

Use one GitHub Project named `DGCC Research`.

Columns:

```text
Backlog → Current → Blocked → Done
```

Fields:

```text
Status
Phase
Decision
Doc
Updated
```

Do not use a `Stage` field. The Project item itself is the milestone.
